# Code of Conduct

Assisted-By Guard is intended to be a practical, respectful project for maintainers and contributors working through AI-assistance disclosure and DCO-style accountability.

## Expected Behavior

- Be respectful and constructive.
- Assume good faith when discussing policy, tooling, or contribution mistakes.
- Keep feedback focused on behavior, documentation, tests, and repository policy.
- Respect that contributors may have different experience levels with DCO, SPDX, GitHub Actions, or AI-assistance disclosure.

## Unacceptable Behavior

- Harassment, threats, or personal attacks.
- Discriminatory language or conduct.
- Publicly shaming contributors for tool use, mistakes, or disclosure questions.
- Posting secrets, credentials, private data, or exploit details in public threads.

## Enforcement

Maintainers may edit, hide, lock, or remove comments and may restrict participation when behavior harms the project community. Enforcement should be proportionate, transparent when practical, and focused on keeping the project useful and safe for contributors.

If you need to report a conduct concern, contact a maintainer through GitHub. For security-sensitive reports, see `SECURITY.md`.
