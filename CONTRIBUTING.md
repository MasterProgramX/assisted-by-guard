# Contributing

Thank you for contributing to Assisted-By Guard.

## Human Accountability

Contributors remain responsible for the changes they submit. If this repository requires DCO-style sign-off, use a human `Signed-off-by` trailer in `Name <email>` form.

AI tools may assist with a contribution, but they must not be listed as DCO signers.

## AI Assistance Disclosure

If an AI tool materially assisted your contribution, disclose it with an accepted trailer such as:

```text
Assisted-by: GitHub Copilot
```

Disclosure is about transparency and maintainability. Assisted-By Guard is not an AI detector and not an AI PR reviewer.

See [DCO and AI contribution guidance](docs/DCO_AND_AI.md) for more detail.
